# 🎸 SaidGila Chord Library

A comprehensive, modern guitar chord reference library featuring **200+ chord voicings** across all 12 root notes, with interactive filtering and visual chord diagrams.

![SaidGila Chord Library](https://img.shields.io/badge/version-1.0.0-gold) ![HTML](https://img.shields.io/badge/HTML-5-orange) ![CSS](https://img.shields.io/badge/CSS-3-blue) ![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)

---

## ✨ Features

- **200+ chord voicings** — Major, Minor, Dominant 7th, Major 7th, Minor 7th, Sus, Add, Extended (9th/11th/13th), Diminished, Augmented, Power Chords, and Special voicings
- **12 root notes** — C, C#, D, D#, E, F, F#, G, G#, A, A#, B
- **Interactive chord diagrams** — SVG fretboard renderings with open strings, muted strings, and barre indicators
- **Smart filtering** — Filter by chord category and root note
- **Live search** — Search by chord name, label, or root note
- **Responsive design** — Optimized for desktop, tablet, and mobile devices
- **Sticky controls** — Always accessible search and filter bar
- **Elegant dark theme** — Inspired by music and modern UI aesthetics
- **Instant feedback** — Real‑time result count and empty state messaging
